# Task Management System

A simple full-stack task management application built with React, TypeScript, Vite, Express, and JWT authentication.

## Features
- Login with default admin user
- Dashboard with task statistics
- Task CRUD operations
- Search, filter, and sorting
- Responsive UI

## Setup

### Backend
```bash
cd backend
npm install
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
