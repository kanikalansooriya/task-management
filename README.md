# Task Management System

A full-stack Task Management System developed as part of a Full Stack Web Developer technical assessment.

This application allows users to securely manage their daily tasks through an interactive dashboard. Users can authenticate, create tasks, update task information, track progress, and view task statistics through a responsive user interface.

The project demonstrates full-stack development practices including frontend development, backend API design, database integration, authentication, and responsive UI implementation.


## Project Overview

### Main Features

- User authentication with JWT
- Secure login functionality
- Dashboard with task statistics
- Create, view, update, and delete tasks
- Task status management
- Task priority management
- Due date tracking
- Recent task overview
- Responsive design
- Dark and light theme support
- Protected routes for authenticated users
